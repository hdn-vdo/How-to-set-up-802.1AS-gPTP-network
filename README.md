# SW2000R-TSN-and-M12X-TSN-Switch
SW2000 Serial100/1000BASE-T RJ45 or M12X-code TSN Switch,  includes 8 channels of 100/1000BASE-T standard Ethernet RJ45, 3channel 1/2.5/5/10G SFP+ and  it can achieve multi-channel switching of Automotive Gateway, AVB application, gPTP boundary clock (BC), gPTP clock bridge.
